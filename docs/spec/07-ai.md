### F1：AI 插件注册机制

```python
# proscore/ai/__init__.py

_providers: dict[str, 'AIProvider'] = {}

def register(name: str, provider: 'AIProvider'):
    _providers[name] = provider

def get_provider(name: str | None = None) -> 'AIProvider':
    if name is None:
        name = list(_providers.keys())[0]
    return _providers[name]

class AIProvider(ABC):
    """AI 插件抽象基类"""
    @abstractmethod
    def chat(self, messages: list[dict], **kwargs) -> str:
        """LLM 对话"""
        ...

    @abstractmethod
    def embed(self, texts: list[str]) -> list[list[float]]:
        """文本 Embedding"""
        ...

    @property
    @abstractmethod
    def name(self) -> str:
        ...
```

### F2：NL2Code 引擎

```python
class NL2Code:
    """自然语言转 proscore 代码"""
    def __init__(self, provider: str | None = None):
        self.provider = get_provider(provider)

    def generate(self, query: str, context: dict | None = None) -> str:
        """将自然语言转为 proscore 代码字符串
        例如: "用卡方分箱8箱跑双向迭代" → "ProScore().bin(method='chi', n_bins=8).select(method='stepwise')"
        """
        prompt = self._build_prompt(query, context)
        code = self.provider.chat([{"role": "user", "content": prompt}])
        return self._validate_code(code)

    def execute(self, query: str, data: pd.DataFrame) -> Any:
        """生成并执行代码"""
        code = self.generate(query)
        return eval(code)
```

### YAML 知识库格式

```yaml

features:
  - name: age
    chn_name: 年龄
    direction: 2          # 1=递增,2=递减,3=U型,4=倒U型
    source: credit_report
    description: 年龄越大，违约率越低

  - name: income
    chn_name: 收入
    direction: 2
    source: application
    description: 收入越高，违约率越低

  - name: inq_last_6mths
    chn_name: 近6月查询次数
    direction: 1          # 递增
    source: credit_report
```

## 降级策略

```
LLM 可用 ────────→ LLM 语义分析
     ↓ 不可用
Embedding 可用 ──→ Embedding 相似度匹配
     ↓ 不可用
规则匹配 ────────→ 知识库查表
     ↓ 不可用
无 AI 降级 ──────→ 纯数据驱动（自动单调性检测）
```
